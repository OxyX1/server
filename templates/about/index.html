possum CS

Possum CS is a secure and customizable credential system built from scratch using Python. It aims to provide a simple, yet effective way to manage user authentication and authorization, supporting various types of credentials, including usernames, passwords, and other authentication methods.

This project was developed with security best practices in mind to ensure data protection and user privacy.
